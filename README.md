# Employee Polls Web App

## Overview

This app allows HR to create polls for employees to post and respond to scenario questions. Employees vote on responses and the dashboard lists employees by the number of polls created and answered.

## Run app

Clone this repo

Make sure you have node version 14 or higher installed

`cd` into your root project and run:

- `npm install`
- `npm start` to start app
- `npm run test` to run testing
